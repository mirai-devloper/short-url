# 短縮URLサービスの作成

## 要件

- サーバーは Netlify
- データベースは supabase
- フロントエンドは React
- APIはNext.jsのAPIルート
- 使用するドメイン名は `s.cc5.me`
- 認証は不要
- ユーザーはURLを入力して、短縮URLを生成できる
- 生成された短縮URLは、`s.cc5.me/` の後にランダムな文字列が付与される（最大8文字）
- 短縮URLの保持期間を設定できるようにする（3日間、7日間、15日間、30日間、60日間、90日間まで）
- 指定期間が過ぎたらデータベースから自動的に削除する
- 削除された短縮URLは、アクセスできなくなる
- 削除された短縮URLは、再生成できない（同じ文字列は生成されない）
